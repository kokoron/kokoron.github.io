初めてのPhoenix

November 12, 2015 by Kokoro

Phoenix elixir

Phoenix

友達にWebの製作を頼まれたので、せっかくなので何か新しい言語とWebフレームワーク使ってみようということで、elixirとPhoenixを始めて触ってみた。 とりあえず elixir のサイトをざっくり読んでみて、なんとなくわかったつもりでPhoenix入れてみる。

インストール

$ brew update
$ brew install elixir

とりあえず Homebrew で elixir をインストール （出典）

$ mix local.hex

Hexというものを入れる必要があるらしい

$ mix archive.install https://github.com/phoenixframework/phoenix/releases/download/v1.0.3/phoenix_new-1.0.3.ez

Phoenixのインストール

nodeも入れておく必要があるとのこと

（出典）

プロジェクト作成

$ mix phoenix.new /path/to/project --database mysql

今回はMySQLを使うので --database mysql をつけます。

Fetch and install dependencies? [Yn] y
* running npm install && node node_modules/brunch/bin/brunch build
* running mix deps.get

...

We are all set! Run your Phoenix application:

    $ cd project
    $ mix ecto.create
    $ mix phoenix.server

You can also run your app inside IEx (Interactive Elixir) as:

    $ iex -S mix phoenix.server

依存関係をインストールする？と聞かれるので y でインストールします。

$ cd project
$ mix ecto.create
$ mix phoenix.server

あとはメッセージに出ているまんま実行します。

http://localhost:4000 をブラウザで開くと、お〜！動いてる〜＼(^o^)／

続く！
